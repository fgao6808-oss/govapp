## 第一步：检查/安装 Node.js

打开终端（Mac：Command+空格 → 输入"终端"→ 回车）：

```
node -v
```

如果显示版本号（如 v18.0.0）→ 已安装，跳到第二步  
如果显示 command not found → 请访问 https://nodejs.org 下载安装 LTS 版本

---

## 第二步：解压并安装项目

```bash
# 1. 解压下载的 govapp.zip，进入文件夹
cd ~/Downloads/govapp

# 2. 安装依赖（首次需要，约1-2分钟）
npm install

# 3. 本地运行
npm start
```

看到以下提示说明启动成功：
```
✅ 物料及商品数据治理系统已启动
🌐 本地访问地址: http://localhost:3000
```

打开浏览器，访问 http://localhost:3000 即可使用。

---

## 第三步：部署到外网（Railway.app）

让任何人都可以通过网址访问系统。

### 3.1 注册 GitHub（如已有可跳过）
访问 https://github.com → Sign up → 免费注册

### 3.2 创建代码仓库并上传
1. 登录 GitHub → 右上角 "+" → "New repository"
2. 仓库名填 `govapp` → Create repository
3. 在终端执行：
```bash
cd ~/Downloads/govapp
git init
git add .
git commit -m "初始提交"
git remote add origin https://github.com/你的用户名/govapp.git
git push -u origin main
```

### 3.3 部署到 Railway
1. 访问 https://railway.app → 用 GitHub 账号登录
2. 点击 "New Project" → "Deploy from GitHub repo"
3. 选择刚才创建的 `govapp` 仓库
4. 等待约 2-3 分钟自动部署完成

### 3.4 设置环境变量（重要！）
在 Railway 项目页面 → Variables → 添加：
```
JWT_SECRET = 任意一串随机字符（如：gov2024xyzabc）
PORT = 3000
```

### 3.5 获取外网地址
部署完成后，Railway 会自动分配地址，在项目页面 → Settings → Domains → 可以看到类似：
```
https://govapp-production-xxxx.up.railway.app
```
将这个地址发给朋友，他们就可以访问了！

## 常见问题

**Q：npm install 出错**  
A：检查网络，或尝试 `npm install --registry https://registry.npmmirror.com`

**Q：Railway 部署失败**  
A：检查 railway.toml 文件是否存在，查看 Railway 的日志定位错误

**Q：数据存储在哪里？**  
A：本地运行时数据存在 `data.db` 文件（SQLite）。Railway 部署后数据存在服务器，重新部署不会丢失。

**Q：忘记密码怎么办？**  
A：本地运行时删除 `data.db` 文件，重启服务即可恢复默认密码 12345a
